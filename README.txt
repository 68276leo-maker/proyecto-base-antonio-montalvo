1. Hacer un fork del proyecto en git
2. Hacer un clon para crear tu repositorio local
3. Crear una rama para crear los ajustes solicitados
4. Desarrollar la operacion de raiz cuadrada en la clase avanzadas
5. Ejecutar add y commit a tu repositorio local
6. Crear el enlace de tu repositorio local con el de github
7. Realizar un push para asumir los cambios
8. Realizar un pull request de los ajustes
9. Esperar la revisión
10. Una vez autorizada acrualizar tu repositorio con la nueva versión
11. Subir los cambios a tu fork en github
